## Introduction
PyTorch implementation for [Noise-Robust Text-to-Image Person Re-Identification via Curriculum-Guided Optimization and Dynamic Re-Ranking](./src/Noise-Robust Text-to-Image Person Re-Identification via Curriculum-Guided Optimization and Dynamic Re-Ranking.pdf) (BICS 2025).  
**CADR** proposes a **Curriculum-guided Alignment & Dynamic Re-ranking** framework to combat noisy correspondence in T2I-ReID.
